# Leaf Venation

Shoutout to Tsoding [for the inspo](https://github.com/tsoding/leaf-venation/)

Implementation of ["Modeling and visualization of leaf venation patterns" paper](https://dl.acm.org/doi/10.1145/1073204.1073251).

![leaf_venation_screenshot](./leaf-venation.png)

# Quick Start

```
cargo run
```

Then press the `space` key to watch veins grow!

Press `r` key to reset.
